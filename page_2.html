if-statements

In PART 1 - paragraaf 2.10 heb je gelezen hoe je met if-statements kan werken. In de main branch ben je al begonnen met een cheatsheet te maken voor C# vs JS overzicht. Voeg de werking en code conventie van een if-statement met een else-if en else toe. Toon hoe dit in C# eruit ziet en in JS.

Kortere if-statement

Je kan een if-statement ook korter maken met een ? en : oplossing. Neem de volgende if-statement: 

                    
                        let studentNumber = prompt('Wat is je studentennummer?', '');
                        let message = '';

                        if (studentNumber < 20000) {
                            message = 'Je bent al lang student bij ons';
                        } else {
                            message = 'Je bent nog maar net student bij ons';
                        }
                        console.log(message);
Je hebt geleerd dat dit ook korter kan: 
                    
                        let studentNumber = prompt('Wat is je studentennummer?', '');
                        let message = (studentNumber < 20000) ? message = 'Je bent al lang student bij ons' : message = 'Je bent nog maar net student bij ons';
                        console.log(message);
Zoals je kan zien ga je van 8 naar 3 regels code. Dit scheelt geheugen, laadtijd en performance.
